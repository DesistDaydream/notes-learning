# 显卡常见配置

## 垂直同步

在不开启垂直同步的情况下，某些游戏可能会无线拉高游戏帧率。开启后，会让游戏的帧率限制到显示器的刷新率，某种程度上，关闭垂直同步会降低显卡的利用率

AI：

**开启垂直同步时：**

- 显卡帧率被限制在显示器刷新率（通常60Hz/144Hz等）
- 如果显卡性能过剩，使用率会**降低**，因为它不需要全力渲染超过刷新率的帧数
- 如果显卡性能刚好或不足，使用率可能保持较高水平

**关闭垂直同步时：**

- 显卡会尽全力渲染，追求最高帧率
- 使用率通常会**提高**，特别是在性能要求高的游戏中
- 可能出现画面撕裂，但帧率更高

**实际影响：**

- 高端显卡玩轻量游戏：开启V-Sync会明显降低使用率
- 中低端显卡玩大型游戏：开启V-Sync对使用率影响较小
- 竞技游戏玩家通常关闭V-Sync以获得更高帧率和更低延迟

现在还有**自适应同步技术**（如G-Sync、FreeSync），可以在防止撕裂的同时避免传统V-Sync的缺点，对显卡使用率的影响更加智能化。

# 芯片厂家
## NVIDIA

### 命名规则

|        | 消费级/企业级     | 系列  | 代系         | 性能等级       | 后缀补充          |
| ------ | ----------- | --- | ---------- | ---------- | ------------- |
| **描述** | 字符串         | 字符串 | 两位数字       | 两位数字       | 字符串           |
| **示例** | GeForce/XXX | RTX | 20/30/etc. | 50/60/etc. | Super/Ti/etc. |

消费级/企业级

- 消费级
    - **GeForce**。通常格式为：`GeForce [系列] [代数][性能等级][后缀]`
- 企业级
    - TODO

系列区分

- **GTX 系列**：较老的游戏显卡系列，不支持硬件光线追踪
- **RTX 系列**：支持实时光线追踪和DLSS技术的新系列

代系。没啥特殊含义，数字越大，芯片越新。

- **20** # 2018-09-20 发布。
- **30**
- **40** # 2022-10-12 发布。23 年 6 月出的 4060
- **50** # 2025-01-06 发布

性能等级数字

- **50**：入门级
- **60**：主流级
- **70**：中高端
- **80**：高端
- **90**：旗舰级

常见后缀

- **Ti**（Titanium）：性能增强版，介于两个等级之间
- **SUPER**：性能提升版本。通常结尾标个 s，e.g. 4060s
- **Mobile/M**：笔记本版本（现在笔记本版本已不再使用M后缀）
- **Max-Q**：笔记本低功耗版本
- **LE**：性能削减版（较少见）

性能通常 Super > Ti > 无后缀

示例解读

- RTX 4070 Ti：40 系列，中高端，增强版
- GTX 1660 SUPER：16 系列，主流级，提升版
- RTX 4090：40 系列旗舰

## AMD

### 命名规则

|        | 品牌        | 代系   | 性能等级 | 细分型号 | 后缀补充 |
| ------ | --------- | ---- | ---- | ---- | ---- |
| **描述** | 字符串       | 一位数字 | 一位数字 | 二位数字 | 字符串  |
| **示例** | Radeon RX | 7    | 8    | 00   | XTX  |

代系

- **7** # 2022-12-13 发布
- **8**
- **9**

性能等级数字

- **4/5**：入门级
- **6**：主流级
- **7**：中高端
- **8/9**：高端旗舰

常见后缀

- **XTX**：最高性能版本
- **XT**：高性能版本
- **无后缀**：标准版本
- **M**：移动版（笔记本）

示例解读

- RX 7900 XTX：7000系列旗舰，最高性能版
- RX 7600：7000系列主流级
- RX 6750 XT：6000系列中高端，高性能版
