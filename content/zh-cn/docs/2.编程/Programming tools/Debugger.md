# 概述

> 参考：
>
> - [Wiki, Debugger](https://en.wikipedia.org/wiki/Debugger)

**Debugger(调试器)** 是一种计算机程序，用于**测试**和**调试** target(目标) 程序。调试器的常见功能包括能够使用 **[breakpoints](https://en.wikipedia.org/wiki/Breakpoint)(断点)** 来运行或暂停目标程序、**[单步执行](https://en.wikipedia.org/wiki/Stepping_(debugging))** 代码，以及显示或修改内存、CPU寄存器和堆栈帧的内容。
