# 概述

> 参考：
> 
> - [Wiki，Cipher](https://en.wikipedia.org/wiki/Cipher)

**Cipher** 在密码学中表示一种用于执行加密或解密的 **algorithm(算法)**。Cipher 不太好翻译中文，password 是密码，Cipher 可以理解为一套**密码系统**，i.e. 一系列可以作为过程遵循的明确定义的步骤

> 注意：cipher 和 cypher 是同一个意思，两种不同的拼写方法
