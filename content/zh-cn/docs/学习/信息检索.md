## 搜索运算符

- `""` # 限定关键词
- **intitle** # 限定标题
- **allintitle** # 限定标题多个关键字
- **intext** # 限定内容关键词
- **inurl** # 限定网址关键词
- **site** # 限定网址来源
- **imagesize** # 限定图片尺寸
- **filetype** # 限定文件格式
