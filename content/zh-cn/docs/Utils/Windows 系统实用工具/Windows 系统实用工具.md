- http://www.uderzo.it/main_products/space_sniffer

- https://github.com/Chuyu-Team/Dism-Multi-language

## 内存管理

**MemReduct**

- https://github.com/henrypp/memreduct
- 轻量级实时内存管理应用程序，用于监视和清理计算机上的系统内存。
