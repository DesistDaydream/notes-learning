# 概述

> 参考：
> 
> - [官方文档，变量](https://gohugo.io/variables/)

Hugo 的模板具有上下文感知能力，让我们在创建网站视图时，可以使用大量的变量。

比如获取当前页面的类型、种类、创建时间、修改时间，我们在配置文件中的很多定义也会被 Hugo 感知到并使用在模板中。

Hugo 的变量用法与 Go 模板变量的用法相同。
