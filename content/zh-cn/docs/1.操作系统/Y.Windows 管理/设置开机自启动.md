# 开机后运行 Powershell 脚本

编写 Powershell 脚本，X.ps1，并将 X.ps1 的快捷方式放到 `%appdata%\Microsoft\Windows\Start Menu\Programs\Startup\ 目录下

设置 .ps1 后缀的默认应用为 Powershell(比如 pwsh、powershell 等)
