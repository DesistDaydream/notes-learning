# [list](https://github.com/libvirt/libvirt/blob/master/docs/manpages/virsh.rst#list) # 列出所有正在运行的 domain

## Syntax(语法)

**list \[OPTIONS]**
OPTIONS

- --all # 列出活动和非活动的所有 domain
- --title # 显示 domain 的 title
- --name # 仅显示虚拟机名称

EXAMPLE

- virsh list #列出所有正在运行的 domain
