# Terminal 增强工具

## tmux

> 参考：
> - [GitHub 项目，tmux/tmux](https://github.com/tmux/tmux)

**terminal multiplexer(终端多路复用，简称 tmux)**，它允许从单个屏幕创建、访问和控制多个终端。 tmux 可能会从屏幕上分离并继续在后台运行，然后再重新连接。
