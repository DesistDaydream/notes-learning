# 概述

> 参考：
>
> - [官方文档，仪表盘 - 变量](https://grafana.com/docs/grafana/latest/dashboards/variables/)

可以人为添加如下几类变量

- **Query** # 变量值从数据源查询中获取
- **Custom** # 手动定义一个变量的值
- **Text box** # 定义一个文本框变量，用户可以在其中输入任意字符串作为变量的值
- **Constant** # 定义一个隐藏常量变量，对于要共享的仪表板中的指标前缀很有用。TODO: 没看懂有啥用
