# 概述
> 参考：

使用 Edge 的用户常会用 newibing 的AI , 但当用了浏览器自带的搜索时，会访问 cn.bing.com 这时BING会把你的位置定位在中国，你再打开 bing.com , 会发现“聊天” 消失了，最筒单的恢复方法是 在浏览器上输入 sg.bing.com 就切回新加坡地点了, 页面上的“聊天”也会重新出现.
